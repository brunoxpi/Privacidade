# Política de Privacidade — Anova

Página pública da Política de Privacidade da **Anova Financial Technologies Ltda.**

## Deploy via Vercel

1. **vercel.com** → login com GitHub → **Add New → Project**
2. Selecione `brunoxpi/PoliticaDePrivacidade`
3. **Framework Preset:** `Other`
4. Clique **Deploy**

## Domínio customizado

Na Vercel: **Settings → Domains** → `privacidade.anovainvestimentos.com.br`

No DNS: CNAME `privacidade` → `cname.vercel-dns.com`

## Atualizar

Edite `index.html` → commit → push → deploy automático.
